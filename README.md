<h2>Clone Project</h2>


```bash
  # Clone the repository
  ❯ git clone https://github.com/rogeriosouz/projeto-ignite-lab.git

  # Enter directory
  ❯ cd netflix-clone_TMDb-API
```

**Using yarn**

```bash
  # Install the dependencies
  ❯ yarn

  # Start the project
  ❯ yarn dev:server
```

**Using npm**

```bash
  # Install the dependencies
  ❯ npm install

  # Start the project
  ❯ npm start
```
